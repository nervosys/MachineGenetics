# I/O and Files

## Reading a file

```rdx
u std.fs

+f main() / io {
    // Read entire file to string
    v content = fs.read("README.md")?
    p"File has {content.len()} bytes"

    // Read as bytes
    v bytes = fs.read_bytes("image.png")?
}
```

## Writing a file

```rdx
u std.fs

+f main() / io {
    fs.write("output.txt", "Hello, world!")?

    // Append
    fs.append("log.txt", "new entry\n")?
}
```

## Buffered I/O

For large files, use buffered readers and writers:

```rdx
u std.io.{File, BufReader, BufWriter, Read, Write}

+f process_large_file(path: &s) / io {
    v file = File.open(path)?
    v reader = BufReader.new(file)

    m line = s.new()
    loop {
        v bytes_read = reader.read_line(&!line)?
        ? bytes_read == 0 { break }
        process(&line)
        line.clear()
    }
}
```

## The Read and Write traits

Implement `Read` or `Write` for custom types:

```rdx
u std.io.{Read, Write}

S Counter {
    count: usize,
}

I Read ~ Counter {
    +f read(&!self, buf: &![u8]) -> R[usize, IoError] / io {
        v data = f"{self.count}".as_bytes()
        v len = data.len().min(buf.len())
        buf[..len].copy_from_slice(&data[..len])
        self.count += 1
        Ok(len)
    }
}
```

## Directory operations

```rdx
u std.fs

+f main() / io {
    // Create directories
    fs.create_dir("output")?
    fs.create_dir_all("deep/nested/path")?

    // List directory contents
    v entries = fs.read_dir("src")?
    @ entry : entries {
        p"{entry.name()} - {entry.file_type()}"
    }

    // Recursive traversal
    v walker = fs.walk("project")
    @ entry : walker {
        ? entry.is_file() {
            p"File: {entry.path()}"
        }
    }

    // File metadata
    v meta = fs.metadata("main.rdx")?
    p"Size: {meta.size()} bytes"
    p"Modified: {meta.modified()}"
}
```

## Standard streams

```rdx
u std.io.{stdin, stdout, stderr}

+f main() / io {
    // Read from stdin
    v input = stdin().read_line()?

    // Write to stdout explicitly
    stdout().write("prompt> ")?
    stdout().flush()?

    // Write to stderr
    stderr().write("warning: something happened\n")?
}
```
