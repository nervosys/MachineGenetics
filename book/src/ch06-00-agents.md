# Agents & Swarms

Redox is the first systems language with **agent primitives in the standard
library**. While other languages treat multi-agent coordination as an
application-level concern, Redox makes it a first-class language feature.

This chapter covers:

- **The Agent trait** — defining agent behavior
- **Swarm orchestration** — coordinating groups of agents
- **Messages & communication** — inter-agent messaging
- **Capabilities & Leases** — permission-bounded agent execution
